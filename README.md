# Text-to-Video

A small script that shows the **visible text** of a webpage **sentence by sentence** on a canvas (and can **record + download** it as a video)

## How to run

1) Install + build:
   
```bash
npm install
npm run build
```

1) Open any webpage → DevTools **Console** → paste:
- `dist/text-to-movie.js`

It runs automatically.

## Config
All config is in: `src/config/config.js`


## Browser support
-  Chrome **121+**
-  Safari **17+**
